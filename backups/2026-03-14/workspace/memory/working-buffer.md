## 什么是危险区

当上下文使用率超过 **60%** 时，系统进入"危险区"。
此时任何对话都可能因上下文压缩而丢失。

**本文件的作用：**
- 在危险区记录每一次人类消息和代理响应
- 即使 SESSION-STATE.md 更新失败，本文件仍有记录
- 上下文压缩后可从此文件恢复状态

```markdown
[1-2句响应摘要 + 关键细节]
```
