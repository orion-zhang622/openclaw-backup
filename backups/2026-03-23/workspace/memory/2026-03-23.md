## 记忆系统修复

### 问题
`memory_stats` 返回 0，无法显示记忆数量

### 根因
- 当 `agentId` 为 undefined 时，`getAccessibleScopes()` 返回 `['global']`
- 但实际数据 scope 是 `agent:main`
- `stats()` 和 `list()` 方法应用了 scope 过滤，导致无匹配数据

### 修复
修改 `/root/.openclaw/workspace/node_modules/memory-lancedb-pro/src/store.ts`:
- 第 696 行和第 744 行
- 当 scope filter 只有 `['global']` 时，绕过过滤

### 结果
- 现在正确显示 131 条记忆 ✓
