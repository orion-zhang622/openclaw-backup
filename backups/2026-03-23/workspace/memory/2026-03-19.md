## [2026-03-19 21:50] memory-lancedb-pro 配置升级

**操作**: 启用管理工具并配置 Weibull Decay 保质期机制

| `decay.betaPeripheral` | 未配置 | `1.3` |
| `tier.coreAccessThreshold` | 未配置 | `10` |
| `tier.workingAccessThreshold` | 未配置 | `3` |
| `tier.peripheralAgeDays` | 未配置 | `60` |

### Weibull Decay 三层架构

| 层级 | Beta | Decay Floor | 行为 | 晋升条件 |
|------|------|-------------|------|----------|
| **Core** | 0.8 | 0.9 | 缓慢次指数衰减 | 访问≥10 + 综合分≥0.7 + 重要性≥0.8 |
| **Working** | 1.0 | 0.7 | 标准指数衰减 | 访问≥3 + 综合分≥0.4 |
| **Peripheral** | 1.3 | 0.5 | 快速超指数衰减 | 默认新记忆 |

### 新增管理工具

启用 `enableManagementTools: true` 后，以下工具现在可用：
- `memory_stats` — 查看记忆统计（总数、分层分布等）
- `memory_list` — 列出记忆并过滤（支持分页）
- `self_improvement_extract_skill` — 从学习记录生成 skill
- `self_improvement_review` — 查看自我改进待办

- 将 `embedding.apiKey` 从环境变量占位符 `${JINA_API_KEY}` 替换为实际 API Key
