## 2026-03-22 21:44 微信公众号自动化代理配置

### 问题
微信 API 调用失败：`invalid ip not in whitelist`

### 解决方案
配置阿里云服务器作为代理出口：

1. **阿里云服务器部署 tinyproxy**
   - 监听端口：8081
   - 出口 IP：115.190.51.253
   - 已修复 DNS 和镜像源问题

2. **本地 frpc 配置**
   ```toml
   [[proxies]]
   name = "claw-a-wechat-proxy"
   type = "tcp"
   localIP = "127.0.0.1"
   localPort = 8080
   remotePort = 8081
   ```

3. **微信 IP 白名单**
   需要添加：115.190.51.253（非 114.215.210.253）

### 技术要点
- frp tcp 代理是反向的（阿里云→本地），不是正向代理
- 直接使用 `http://114.215.210.253:8081` 作为代理地址
- 阿里云 ECS 公网 IP 通常固定，但重启/重建可能变化
