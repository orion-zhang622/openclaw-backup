## Scrapling 网页抓取库调研

### 核心能力
- **自适应选择器**: 网站改版后仍能定位元素（auto_save + adaptive）
- **反爬绕过**: StealthyFetcher 可处理 Cloudflare Turnstile
- **高性能解析**: 比 BeautifulSoup 更快

### 与现有工具整合方案
- **方案A（推荐）**: agent-browser 负责交互+截图，Scrapling Parser 负责解析
- 两者不冲突，Parser 模块不启动浏览器
- 不能替代 agent-browser 的截图和登录功能

### IP风控应对
- Scrapling 提供代理轮换功能（需自备代理IP）
- 可降低请求频率、行为伪装
- 不能解决 IP 已被封的问题，需购买代理服务
